# 💱 Currency Converter

Real-time valyuta konvertori - zamonaviy web ilova

## 📖 Loyiha haqida

Bu loyiha real-time valyuta kurslarini konvertatsiya qilish uchun yaratilgan zamonaviy web ilova. Foydalanuvchilarga qulay interfeys, dark/light mode, sevimli valyuta juftliklarini saqlash va kurs o'zgarishlarini kuzatish imkoniyatlarini taqdim etadi.

## 🎯 Maqsad

- Foydalanuvchilarga real-time valyuta kurslarini taqdim etish
- Qulay va zamonaviy interfeys yaratish
- Mobil qurilmalarda ham mukammal ishlash
- Kurs o'zgarishlarini avtomatik kuzatish
